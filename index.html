Historical context: Used by Julius Caesar around 100 BC to protect military communications. ROT13 (shift=13) is its own inverse — applying it twice returns the original text. Both are trivially broken by frequency analysis, making them foundational examples of why substitution alone is insufficient in modern cryptography.

What Is This

A Browser-Based Security Toolkit

Cipher is a collection of five interactive security utilities built entirely from scratch — no frameworks, no external dependencies, no data ever leaves your browser. Every calculation runs locally using native Web APIs. It was developed as a hands-on portfolio project to reinforce the theory studied whilst working towards CompTIA Security+ certification.

The project is intentionally transparent: all source code is visible, readable, and commented so that other learners can follow along, identify mistakes, or build on top of it. Constructive feedback is always welcome.

How to Use It

Getting Started

  1. Select a module from the navigation on the left.
  2. Enter into the provided field.
  3. Click the primary action button to run the tool.
  4. Review the output and any contextual notes below the result.
  5. Use the Copy button to copy results to your clipboard.

All tools work offline — no internet connection required after the page loads.

Concepts Covered

Technical Skills

  • Cryptographic hashing — MD5, SHA-1, SHA-256 via the Web Crypto API; understanding collision resistance and real-world use cases.
  • Password entropy — Based on NIST SP 800-63B guidelines; charset analysis and crack-time estimation at 10 billion guesses per second.
  • Classical ciphers — Caesar cipher and ROT13; substitution weaknesses and frequency analysis.
  • Encoding formats — Base64, URL encoding, hexadecimal, binary. Encoding ≠ encryption.
  • URL structure auditing — Protocol inspection, homograph detection, open redirect indicators, and suspicious parameter names.

Stack

Built With

  • Vanilla HTML5 / CSS3 / JavaScript (ES2020+) — no build tools or frameworks
  • Web Crypto API — SubtleCrypto for SHA-1 and SHA-256
  • Canvas API — procedural star field background
  • Orbitron + IBM Plex Mono — loaded via Google Fonts
  • Semantic HTML + ARIA — accessible tab interface and live regions
  • Zero external dependencies — fully self-contained single file
